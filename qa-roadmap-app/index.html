简历项目写法

接口自动化测试框架项目

基于 Python + pytest + requests + YAML + Allure 搭建接口自动化测试框架，支持测试数据驱动、公共请求封装、断言校验、测试报告生成，可接入 Jenkins 实现持续集成。

  • 封装统一请求方法，减少重复代码，提高用例可维护性。
  • 使用 YAML 管理接口路径、请求参数和预期结果，实现数据驱动。
  • 通过 pytest 参数化执行多组用例，输出 Allure 可视化报告。
  • 对登录、查询、异常参数等接口场景进行自动化覆盖。

你要能讲清楚的亮点

框架分层、断言策略、token 管理、失败定位、日志记录、测试数据维护、持续集成触发方式。
